## Job Placements Portal (MERN)

### Features
- Role-based authentication (Candidate / Recruiter)
- Job posting & applications
- Resume management
- Application tracking & approval
- Dashboards with analytics

### Tech Stack
- React + Redux Toolkit
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication

### How to Run
1. Clone repo
2. Install client & server dependencies
3. Setup .env
4. Run both servers
